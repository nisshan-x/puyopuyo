Puyo puyo game

# Requirement

- Python3
- pygame
- Numpy
- Joystick (option)

# Installation

## pygame

```bash
pip3 install pygame
```

# Usage

```bash
python3 main.py
```

## Key input

| Key   | PS3 Joystick      | Function               |
| ----- | ----------------- | ---------------------- |
| LEFT  | LEFT              | Move left              |
| RIGHT | RIGHT             | Move right             |
| DOWN  | DOWN              | Fall fast              |
| Z     | Cross             | Rotate left            |
| X     | Circle            | Rotate Right           |
| R     | START             | Restart game           |
| P     | SELECT            | Pause and resume game  |
| Q     | L + R             | Quit                   |
| W     | Square + Triangle | Puyos appear randomly! |
